Things I've built

Featured Projects

01

Classification of Simulated Microgravity-Induced Muscle Atrophy

using Ensemble Learning

Built an ensemble model (Random Forest, SVM, XGBoost) to classify muscle atrophy from NASA GeneLab gene-expression data. Applied t-test and Boruta feature selection to reduce 70,000 features to 60. Achieved high AUC & F1 scores with SHAP interpretation and GO/KEGG enrichment validation.

NASA GeneLab Random Forest SVM XGBoost SHAP Gene Expression
Machine Learning
02

Remaining Useful Life Prediction for Aircraft Engines

using GRU

Developed a GRU-based model to predict Remaining Useful Life (RUL) of aircraft engines on the NASA C-MAPSS dataset. Integrated monotonic constraints and uncertainty-aware techniques for reliable predictions. Built a Health Index (Normal/Mild/Severe) and deployed a Gradio interface for real-time insights.

Deep Learning GRU Time Series Predictive Maintenance NASA C-MAPSS Gradio
Deep Learning
03

Smoke Level Monitoring & Alerting in Automobiles

using IoT

Developed a real-time embedded system to detect CO, NO₂, and HC emission levels using MQ sensors and Arduino. Integrated GSM (SIM900) module and buzzer alerts to notify drivers and authorities when emissions exceed safe thresholds.

IoT Arduino MQ Sensors GSM SIM900 Embedded Systems
